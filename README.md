# Pickleball Scoreboard

Petite application front-end (HTML/CSS/JS vanilla) pour suivre un match de pickleball.

## Fonctionnalités

- Comptage des points des deux équipes avec boutons +1/-1
- Indicateur de service actuel
- Possibilité de renommer les équipes via une boîte de dialogue native
- Configuration du nombre de points pour gagner et de l'écart minimum
- Indication "balle de match" / annonce du gagnant
- Remise à zéro rapide pour démarrer une nouvelle partie

## Démarrage

Aucun build n'est nécessaire. Il suffit d'ouvrir `index.html` dans votre navigateur préféré ou d'utiliser une extension type "Live Server" pour bénéficier du rechargement automatique.
